About the Project

This Final Year project is a Network Intrusion Detection System (NIDS) that uses machine learning algorithms to detect and prevent cyber attacks. The system continuously monitors network traffic, analyzes it in real-time, and uses Machine Learning powered algorithms thought continuously by Threat intel feeds, to identify and respond to potential threats. The goal of this project is to demonstrate the effectiveness of machine learning in enhancing network security and protecting against cyber threats.

Key Features

Real-Time Monitoring

Continuously analyzes network traffic for anomalies.

AI-Powered Threat Detection

Uses ML models (SVM, LSTM, clustering) to detect attacks.

Automated Alerts & Reporting

Instant notifications and detailed incident reports.
